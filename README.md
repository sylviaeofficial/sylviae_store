# Sylviae Store

A high-end, ultra-minimalist personal brand website for www.sylviae.store.

## Tech Stack
- Pure HTML5
- Modern CSS3 (CSS Variables, Flex/Grid layouts, Glassmorphism design concepts)
- Vanilla JavaScript (Custom cursor, animations, intersection observer logic)
- Locally hosted Inter Variable Typography

## Deployment to GitHub Pages
1. Push this directory to the `main` branch of your GitHub repository.
2. Go to the repository **Settings**.
3. Navigate to **Pages** on the left sidebar.
4. Under **Source**, select `Deploy from a branch`.
5. Under **Branch**, select `main` and the root folder `/(root)`.
6. Click **Save**. Note: check the Actions tab to monitor the build and deployment progress!
7. Once successfully deployed, configure your custom domain `www.sylviae.store` in the "Custom domain" field.

## Features Let Down
- **Nature Distilled + Frosted Touch**: A design concept focusing on soothing, organic colors alongside precise, glassmorphic layout treatments.
- **Asymmetrical layout**: Creates natural eye movements and appreciates the beauty of balanced "white space".
- **Magnetic cursor trails**: Added micro-interactions that elevate UI. 
- **Kinetic Typography**: An organic ripple typing feeling on the main title hero header.
